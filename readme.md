### redis KEY定义
| key名                  | 含义             |      |
| ---------------------- | ---------------- | ---- |
| groupInfo:groupId      | 存储group信息    | hash |
| groupMsg:groupId       | 存储群消息       | hash |
| groupsForUser:username | 存储用户所在群聊 | hash |
|                        |                  |      |
